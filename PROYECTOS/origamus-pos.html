Descripción del Proyecto

Biblioteca App es una solución de escritorio robusta diseñada para simplificar la gestión de inventario en bibliotecas. Permite a los administradores llevar un registro de préstamos, devoluciones, gestionar usuarios y catalogar nuevos libros de forma intuitiva.

El objetivo principal era crear una herramienta rápida, segura y que no dependiera de una conexión a internet constante, ideal para escuelas o instituciones con infraestructura de red limitada.

Detalles del Proyecto

  • Tipo: Plataforma Web / Aplicación
